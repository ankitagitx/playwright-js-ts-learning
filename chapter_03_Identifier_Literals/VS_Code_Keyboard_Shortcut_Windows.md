## Tips for Windows Users

- **Ctrl** is the primary modifier key on Windows (same as Linux)
- **Alt** is the Alternative key on Windows keyboard
- **Shift** is the Shift key on Windows keyboard
- You can customize these shortcuts in VS Code Settings: `Ctrl + K + Ctrl + S`
- Many shortcuts can be combined for more advanced functionality
- Memorize the most common ones first: `Ctrl + P`, `Ctrl + F`, `Ctrl + S`, `Ctrl + Z`
- Some shortcuts may conflict with Windows system shortcuts - customize in VS Code if needed

## Frequently Used Shortcuts Cheat Sheet

```
File Operations:    Ctrl + N (new), Ctrl + O (open), Ctrl + S (save)
Search:            Ctrl + F (find), Ctrl + H (replace)
Navigation:        Ctrl + P (go to file), Ctrl + Shift + O (symbols)
Editing:           Ctrl + D (select), Ctrl + / (comment), Ctrl + K + Ctrl + F (format)
Terminal:          Ctrl + ` (toggle), Ctrl + Shift + ` (new)
Debug:             F5 (start), F10 (step over), F9 (breakpoint)
Version Control:   Ctrl + Shift + G (open source control)
```
