## Tips for macOS Users

- **Cmd** is the primary modifier key on macOS (instead of Ctrl on Windows/Linux)
- **Alt** is the Option key (⌥) on macOS keyboard
- **Ctrl** is the Control key (⌃) on macOS keyboard
- **Shift** is the Shift key (⇧) on macOS keyboard
- You can customize these shortcuts in VS Code Settings: `Cmd + K + Cmd + S`
- Many shortcuts can be combined for more advanced functionality
- Memorize the most common ones first: `Cmd + P`, `Cmd + F`, `Cmd + S`, `Cmd + Z`

## Frequently Used Shortcuts Cheat Sheet

```
File Operations:    Cmd + N (new), Cmd + O (open), Cmd + S (save)
Search:            Cmd + F (find), Cmd + Alt + F (replace)
Navigation:        Cmd + P (go to file), Cmd + Shift + O (symbols)
Editing:           Cmd + D (select), Cmd + / (comment), Cmd + K + Cmd + F (format)
Terminal:          Ctrl + ` (toggle), Cmd + Shift + ` (new)
Debug:             F5 (start), F10 (step over), F9 (breakpoint)
Version Control:   Ctrl + Shift + G (open source control)
```
